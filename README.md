# Provence Wine Club

Provence Wine Club is a static editorial website for Emilie Blanc's Provence rose wine project in Australia. The site introduces the club, tells the Provence story, and supports search discovery with guide, pairing, press, and knowledge pages.

## Live Site

- Primary domain: https://emilieblanc.wine
- GitHub repository: https://github.com/Mims2DaBee/ProvenceWineClub

## What Is Included

- Homepage for Provence Wine Club
- Emilie Blanc profile and press pages
- Provence rose wine guide
- Rose and seafood pairing guide
- Structured knowledge pages for search and AI discovery
- `sitemap.xml`, `robots.txt`, and `llms.txt`

## SEO Focus

The site is optimized around:

- Provence rose wine in Australia
- Provence Wine Club
- Emilie Blanc wine expert
- Provence rose tastings and events
- Mediterranean food pairings with rose

## Project Structure

```text
assets/                 Images and SVG brand assets
data/                   Structured wine knowledge page
guides/                 Long-form wine education pages
knowledge/              Entity and expert profile pages
pairings/               Food and wine pairing content
index.html              Homepage
emilie-blanc.html       Main expert profile page
press.html              Media and press page
sitemap.xml             Search sitemap
robots.txt              Search and crawler rules
llms.txt                AI discovery summary
```

## Local Preview

Open `index.html` in a browser from this folder. The site is static HTML, CSS, and JavaScript, so it does not require a build step.

## Deployment

Deploy by pushing `main` to GitHub:

```bash
git push origin main
```
